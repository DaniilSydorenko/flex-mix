Introduction

Welcome to Flex-Mix. It is a boilerplate for frontend development based on flex-grid(http://flexboxgrid.com/) and sass boilerplate(https://github.com/HugoGiraudel/sass-boilerplate).

Installation

git clone git@github.com:DaniilSydorenko/flex-mix.git

Bower

bower install

or

sudo bower install --allow-root

NPM

sudo npm install
